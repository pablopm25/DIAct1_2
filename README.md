# Actividad 1.1: Gestión de Alumnos (Desarrollo de Interfaces)

Proyecto realizado en Java (Swing) para la gestión de altas de alumnos en un centro educativo.

## Descripción
La aplicación consiste en un formulario gráfico que permite dar de alta alumnos validando sus datos y almacenándolos en memoria.

## Funcionalidades Implementadas
* **Clase Modelo:** Creación de la clase `Alumno` con encapsulación, constructores y métodos específicos.
* **Interfaz Gráfica:** Formulario realizado con `JFrame` y `JTextField`.
* **Validaciones:**
  * Control de que la edad sea un valor numérico.
  * **Array Estático:** Control del límite máximo de 5 alumnos.
  * Mensajes de error visuales (etiqueta roja) según especificaciones.

## Tecnologías
* Java JDK
* Java Swing (Librería gráfica)
* IDE: Visual Studio Code
